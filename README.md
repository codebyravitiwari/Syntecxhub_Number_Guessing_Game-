# Number Guessing Game 🕹️

## Overview

A professional CLI-based Number Guessing Game built in Python. Features difficulty levels, attempt limits, persistent high scores, input validation, and a polished terminal interface.

## Quick Start

1. Ensure Python 3.6+ is installed.
2. Navigate to the project: `cd number_guessing_game`
3. Run: `python main.py`

## Features

- **3 Difficulty Levels**: Easy (1-50, 15 attempts), Medium (1-100, 10 attempts), Hard (1-500, 7 attempts)
- **Smart Hints**: Too high / Too low feedback
- **High Score Tracking**: Persistent best scores per difficulty in `scores.json`
- **Input Validation**: Handles invalid inputs gracefully
- **Replay & Exit**: Full game loop
- **Performance Ratings**: Excellent/Good/Average based on attempts
